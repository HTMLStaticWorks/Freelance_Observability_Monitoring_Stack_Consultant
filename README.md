# Freelance Observability & Monitoring Stack Consultant Website

A premium, high-end, responsive multi-page website designed for an **Observability & Monitoring Stack Consultant**. Built using HTML5, CSS3, and Vanilla JavaScript, this project features a modern dark-themed aesthetic with advanced UI components.

## 🚀 Features

- **Premium Design**: Modern aesthetic with glassmorphism, smooth gradients, and micro-animations.
- **Fully Responsive**: Optimized for all devices from mobile (320px) to desktop.
- **Dark Mode & RTL Support**: Built-in support for theme switching and right-to-left languages.
- **Multi-page Layout**:
  - Home Page (Multiple variations)
  - About Us
  - Services & Service Details
  - Portfolio
  - Pricing
  - Contact
  - Authenticated DevOps Dashboard
  - 404 & Coming Soon pages
- **Interactive Components**: 3D tilt effects, animated counters, and dynamic charts (via Chart.js).
- **SEO Optimized**: Semantic HTML5 structure and optimized metadata.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript (Vanilla)
- **Icons**: Font Awesome / Boxicons
- **Fonts**: Google Fonts (Inter, Outfit)
- **Animations**: AOS (Animate on Scroll)
